# RED ANGPOW API SERVER

## Installation

```
$ npm i
```

## Development

To start development

```
$ npm run dev
```

## Production (Local)

To start production environment locally

```
$ npm run start
```

### Local Development

Place `config.env` on root of directory

```
DATABASE=<...>
DATABASE_PASSWORD=<...>
PORT=<...>
```
